Those needed for further development and delivering a final app product, ready for market in an app store:

AWS (Amazon Web Services) for secure cloud platform. Could actually host the entire server infrastructure. Services that might work for us:

  • AWS Cloud tools - see below
  • AWS Amplify - development platform for secure and scalable mobile apps. It includes ready-to-use UI components like sign up and login forms for the app.
  • AWS Amplify Framework to develop UI components
  • AWS Amplify API
  • AWS Amplify Authentication library which provisions Amazon Cognito.
  • AWS/Amazon Cognito (for user accounts) and manage it from AWS Amplify.
  • Amazon CodePipeline and CodeDeploy for Continuous Integration
  • React Native can give cross-platform for iOS and Android and allows push notifications to both.
  • Amazon Pinpoint for push notifications.
  • AWS Amplify DataStore libraries, AWS AppSync (creates flexible and secure APIs which integrate into other AWS services), and Amazon DynamoDB (data storage), for data management
  • Amazon Athena for basic analytics, moving to Amazon Kinesis when a certain amount of users is reached
  • AWS Device Farm for testing
